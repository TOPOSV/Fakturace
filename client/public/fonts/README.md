# Fonts for PDF Generation

This directory contains fonts used for PDF generation with Czech diacritics support.

## Required Font
- **Roboto-Regular.ttf** - Download from: https://fonts.google.com/specimen/Roboto

## Installation
1. Download Roboto-Regular.ttf
2. Place it in this directory  
3. The PDF generator will automatically load it

## Font Conversion
Alternatively, convert the font to base64 using:
https://raw.githack.com/MrRio/jsPDF/master/fontconverter/fontconverter.html
